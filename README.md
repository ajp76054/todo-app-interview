# Routes

API URL is located in the `variables.ts` file and is already imported into the `todos.service.ts` file for you to consume.  The API accepts any of the follow request types in the list.

 - GET	/todos
 - GET	/todos/1
 - POST	/todos
 - PUT	/todos/1
 - PATCH	/todos/1
 - DELETE	/todos/1
